# Nuzdex

Nuzdex is an iOS fan app for looking up battle information in Pokemon games.

Current in-app data focuses on:
- Emerald
- Fire Red
- Leaf Green

## Features
- Switch game from the top-left game bubble.
- Browse wild encounter lists in Routes.
- Browse Gym Leaders and Elite Four in Bosses.
- Browse Rival / Evil Team / other key fights in Battles.
- Starter-dependent battle data where needed.
